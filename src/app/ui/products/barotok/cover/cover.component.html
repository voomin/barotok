커버 대형(Cover Large)
제품설명

바로톡 거름망에 맞게 사용자의 불편함과 개선사항을 분석하여 싱크대 커버를 개발하였습니다.

설거지 할 때마다 커버에 걸리는 음식물을 매번 손으로 털어줘야 하는 불편함을 해소하기 위해 바로톡은 기존의 가이드를 없애 커버에 걸리는 음식물을 최소화하여 사용자가 더 편하게 사용할 수 있도록 리뉴얼하였습니다.

U자형으로 디자인되어 다양한 형태의 음식물이 걸리지 않고 자연스럽게 배출됩니다.
표면 광택으로 오염되는 것을 최소화 하였고 세척이 용이합니다.
  • ※디자인 등록이 되어있는 상품입니다.
제품사양
제품명 싱크대 거름망 커버 대형 (삐에로)
사용규격 배수구 내경 143~145mm
구성품 커버, 리필 거치링 SET
소재 ABS
사용방법


1. 기존에 사용하시던 스테인리스 거름통을 제거해주세요.
2. 거치링에 거름통을 씌워 배수구에 넣어주세요.
3. 커버를 올려주세요.
